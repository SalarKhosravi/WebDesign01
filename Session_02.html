تگ های پایه
<!DOCTYPE> برای مشخص کردن نوع یا نسخه صفحه وب
<html> برای تعریف ریشه و ستون اصلی یک صفحه وب
<head> برای تعریف قسمت هد یک صفحه وب
<meta> برای تعریف متاداده ها در یک صفحه وب
<title> برای تعریف عنوان صفحه در مرورگرها
<body> برای ایجاد بدنه / ستون اصلی یک صفحه وب
<h1> – <h6> برای تعریف تگ های تیتر یا عنوان در صفحه وب
<p> برای ایجاد پاراگراف در یک صفحه وب
<br> برای شکستن متن و رفتن به سطر جدید
<hr> یک خط افقی برای جدا کردن بخش ها از همدیگر
<!–…–> برای تعریف توضیحات در صفحه وب
مربوط به قالب بندی متن
<abbr> برای تعریف مخفف ها در صفحه وب
<address> برای تعریف اطلاعات تماس نویسنده
<b> برای برجسته یا Bold کردن متن ( بدون ارزش محتوایی )
<bdi> برای تعریف جهت متن / محتوا
<bdo> برای برعکس کردن متن
<blockquote> برای تعریف نقل قول های بلند از یک منبع خارجی
<cite> برای تعریف نام یک اثر، کتاب ، فیلم ، موسیقی و…
<code> برای قرار دادن کد ( مثلا دستورات html ) در صفحه وب
<del> برای حذف کردن محتوا از صفحه ( روش خط کشیده میشه )
<dfn> میتوان اصطلاحات رو در یک صفحه وب تعریف کرد
<em> برای تاکید کردن بروی یک محتوا ( دارای ارزش محتوایی )
<i> برای کج / شکسته کردن متن
<ins> یک خط به زیر محتوا میکشد ( دارای ارزش محتوایی )
<kbd> برای تعریف ورودی های صفحه کلید در صفحه وب
<mark> برای رنگی کردن محتوا ( رنگ زرد یا ماژیک فسفری )
<meter> میتوان یک گیج ( Gauge ) اندازه گیری را ایجاد کرد
<progress> برای نمایش میزان پیشرفت کار ( پروژه و.. )
<q> برای ایجاد نقل قول کوتاه
<rp> برای مشخص کردن علائم نگارشی زبان های شرق آسیا
<rt> برای مشخص کردن علائم نگارشی زبان های شرق آسیا
<ruby> برای مشخص کردن علائم نگارشی زبان های شرق آسیا
<s> برای خط کشیدن بروی یک متن ( یعنی اون متن تاریخ اعتبارش تموم شده )
<samp> برای نمایش خروجی یک برنامه کامپیوتری در صفحه
<small> اندازه یک متن را کمی کوچکتر نمایش میده
<strong> یک متن رو برجسته یا Bold میکنه ( دارای ارزش محتوایی )
<sub> برای ایجاد یک متن پایین نویس
<sup> برای ایجاد یک متن بالا نویس
<pre> برای ایجاد یک محتوا با قالب بندی دلخواه
<template> برای مخفی کردن محتوا در هنگام بارگذاری صفحه
<time> برای تعریف زمان ( تاریخ و ساعت ) در یک صفحه وب
<u> برای خط کشیدن به زیر یک متن ( بدون هیچ ارزش محتوایی )
<var> برای تعریف متغیرها در صفحه وب
<wbr> جهت شکستن متن به سطر جدید
فرم ها و اینپوت ها
<form> برای ایجاد یک یا چندین فرم در صفحه وب
<input> برای قرار دادن اینپوت ها در فرم
<textarea> برای ایجاد یک باکس متن / پیغام
<button> برای ایجاد یک دکمه قابل کلیک
<select> برای ایجاد یک لیست بازشو
<option> خوده آیتم های لیست بازشو را مشخص میکند
<optgroup> برای دسته بندی آیتم های یک لیست بازشو
<label> برای تعریف لیبل برای اینپوت های داخل فرم
<fieldset> برای گروه بندی تگ های داخل فرم
<legend> برای ایجاد یک عنوان برای تگ <fieldset>
<datalist> برای تعریف یک لیست از گزینه های از پیش تعریف شده
<output> برای نمایش خروجی یک عمل انجام شده
مربوط به فریم (قاب یا پنجره ها)
<iframe> برای ایجاد فریم ها در صفحه وب
<embed> برای قرار دادن عکس ، ویدئو، PDF و.. در صفحه
<object> برای قرار دادن عکس ، ویدئو، PDF و.. در صفحه
<param> برای تعریف پارامترهای تگ <object>
مربوط به تصاویر
<img> برای گذاشتن عکس / تصویر در صفحه وب
<map> برای تعریف یک نقشه تصویری ( تصویری که بروی آن تعدادی لینک وجود داشته باشد )
<area> تعریف ناحیه های قابل کلیک در یک نقشه تصویری
<canvas> یک ظرف برای نگهداری اشکل گرافیکی ایجاد شده توسط جاوا اسکریپت
<figure> برای تعریف شکل ها ، تصاویر ، نمودار ها و.. در صفحه
<figcaption> برای ایجاد یک عنوان برای تگ <figure>
<picture> یک ظرف برای نگهداری چندین عکس / تصویر
<svg> یک ظرف برای نگهداری اشکال گرافیکی ایجاد شده توسط SVG
مربوط به رسانه (موزیک و ویدئو)
<audio> برای گذاشتن آهنگ ( فایل صوتی ) در صفحه وب
<video> برای گذاشتن ویدئو در صفحه وب
<track> برای ایجاد محتوای متنی یک رسانه ( مثلا زیرنویس ویدئو )
<source> برای مشخص کردن مسیر فایل برای تگ های چندرسانه
مربوط به لینک ها
<a> برای تعریف لینک ها در صفحه وب
<link> برای وصل کردن فایل CSS به فایل HTML
<base> برای تعریف آدرس لینک پایه در یک صفحه وب
<nav> برای تعریف منو ها در صفحه
مربوط به لیست ها
<ul> برای ایجاد لیست های نامرتب
<ol> برای ایجاد یک لیست مرتب شده
<li> برای تعریف آیتم های یک لیست مرتب یا نامرتب
<dl> برای ایجاد یک لیست توضیحی
<dt> برای ایجاد آیتم های یک لیست توضیحی
<dd> برای تعریف توضیحات آیتم های یک لیست توضیحی
مربوط به جدول ها
<table> برای ایجاد جدول در صفحه وب
<caption> تعریف کپشن برای یک جدول
<tbody> برای ایجاد بدنه محتوای جدول
<td> برای ایجاد سلول های یک جدول
<tfoot> برای ایجاد فوتر جدول
<th> برای ایجاد ستون های جدول
<thead> برای ایجاد هدر جدول
<tr> برای ایجاد سطرهای یک جدول
<col> برای انتخاب گروهی ستون های جدول
<colgroup> برای انتخاب گروهی ستون های جدول
مربوط به استایل و تگ های معنایی یا Semantic
<style> برای تعریف دستورات CSS درون صفحه وب
<div> ظرفی برای نگهداری دیگر تگ های HTML
<span> یک تگ Inline برای ایجاد محتوای درون خطی
<header> تعریف هدر برای یک بخش خاص یا کل صفحه
<footer> تعریف فوتر برای یک بخش خاص یا کل صفحه
<main> برای ایجاد بخش اصلی در یک صفحه وب
<section> برای تعریف یک بخش / قسمت در یک صفحه وب
<article> برای تعریف مقاله / مطالب در صفحه وب
<aside> برای تعریف محتوای مرتبط با محتوای اصلی سایت
<details> برای ایجاد یک باکس محتوا بصورت بازشو
<dialog> برای ایجاد یک پنجره دیالوگ ( پاپ آپ یا مودال )
<summary> برای ایجاد عنوان برای تگ <details>
<data> برای ایجاد یک محتوای قابل درک برای موتورهای جستجوگر و…
مربوط به اسکریپت ها
<script> برای تعریف / ایجاد دستورات جاوا اسکریپت در صفحه
<noscript> برای تعریف یک متن دلخواه – جهت نمایش به کاربر در صورت عدم پشتیبانی مرورگر کاربر از تگ Script
